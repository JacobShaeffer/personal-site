Fast Tunnel

Is a 2D arcade game I made for Android. The goal of the game is to travel as far as possible down an endless tunnel. It's more fun than that makes it sound, I swear. The tunnel is constantly shifting and moving constantly trying to kill the ship.

It's witten in Java in the Android Studio IDE. I used a game framework called libGDX.

Hotdog

Ever wanted to play Connect-4 on a 3D game board? Yeah, me neither, but I made it anyway.

I made it with WebGL and Three.js for the rendering with a Node.js backend server using Socket.IO for client server communication. All of the game code is Javascript.

Play it here!

Travel Activiy Collection App (Taca)

Is a crossplatform (iOS and Android) app that records the users location and generates "Stops" from this data. This is for the Maricopa Association of Governments and it is for a survey they conduct to improve their roadway models.

It is currently under development, but should be done and live on the Google Play Store by mid 2018. The name will have also changed by then, Taca is a temporary name we use to discuss it.
